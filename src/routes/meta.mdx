# 关于本站如何搭建

<Status type="seed" />

这是花园里的第一颗种子。我决定遵循“Learn in Public”的原则，记录下搭建这个网站的过程。

### 🛠️ 技术栈

*   **框架：** SolidStart
*   **样式：** UnoCSS
*   **内容：** MDX

### 📝 搭建日志

#### 2025-11-19：最小可行性花架 (MVP)

今天我按照“数字花园”的理念，重构了网站的布局。

*   **放弃了大而全的博客设计**，转而采用“侧边栏 + 正文”的双栏布局。
*   **引入了状态指示器**：用 🌱 和 🌲 来区分内容的成熟度。
*   **建立了三个核心页面**：首页（宣言）、Now（当下）、Meta（本站）。

现在的花园看起来还很简陋，但它已经是一个可以“生长”的系统了。每当我有一个新的想法，我只需要新建一个文件，打上 🌱 标签，就可以毫无压力的发布出来。
